# Pokedex - Tomasz Ogrodnik

Aplikacja Pokedex bedzie baza wiedzy na temat wszystkich pokemonów

Aplikacja bedzie zawierać ekrany

### Aplikacja Będize zawierać ekrany:

- Edycje Pokemonów - bedzia sie tam zawierać guziki które bedia zawierać sezon edycje itd

- Detale pokemona - Bedą zawierac wszystkie informacje na temat jednego

- Wszystkie Pokemony - Wszystkie pokemony ze wszystkich edycji
